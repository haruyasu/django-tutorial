## はじめに

今回は、初心者向けのDjangoアプリ開発チュートリアルです。

すぐにDjangoを使ってアプリ開発をしてみたい方向けの内容になっています。

このチュートリアルを実装すると、Webアプリケーションを公開することができます。

シンプルなブログを作って、Djangoの機能を理解していきましょう。

## バージョン

Python：3.7

Django：2.2

※Djangoの最新バージョンは3.0ですが、2系と3系では変更点が多いので、情報が多い2系を使用して下さい。
